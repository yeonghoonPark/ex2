Relative length units 🤩

  • em: (relative to parent element) 선택된 폰트패밀리와는 상관없이, 즉 선택된 폰트와 상관없이 고정된 폰트사이즈를 갖는다. em은 부모의 폰트사이즈의 상대적으로 크기가 계산된다.

  • rem: (relative to root element) 부모에 따라 사이즈가 결정되는 것이 아닌, 즉 root에 따라 사이즈가 결정된다.

  • vw: viewport width

  • vh: viewport height

  • %: 부모요소에 상대적으로 계산된다.
  • parent가 기준: %, em

  • browser가 기준: vw, vh, rem

  • 요소의 width와 height가 기준: %, vw, wh

  • 요소의 font-size가 기준: em, rem
